## Strategic actions (deadline-driven)

### Milestone 1 — Manuscript lock (target: ≥ 3 weeks before 31 Oct)
1. **Word-count pass** to the 5000-word ceiling; the diagram does the heavy lifting so prose
   can be trimmed around it. Confirm figure count/placement against CAA proceedings rules.
2. **Style conversion**: reformat the Springer extended version into CAA author styles
   (single source of truth for shared sections so the two manuscripts stay in sync).
3. **Reference hygiene**: complete the dataset handle/DOI swap to `hdl:20.500.12800/1-6184`
   and the regenerated Zenodo PDF — this is the one cross-issue dependency (#21), so it
   gates the bibliography being final.

### Milestone 2 — Internal review (target: ~2 weeks before)
4. Co-author review via Overleaf (David, Dana); collect comments in one pass, not serially.
5. Verify every external link resolves (Zenodo DOI, dataset handle, repo links) — the same
   link-rot that #21 and #15/#16 are chasing applies here.

### Milestone 3 — Submit
6. **PCI Archaeology** new submission; deposit the proceedings record into the CAA2026 Zenodo
   community. Record the submission ID back in #13.
7. Keep #13 open until acceptance; track the Springer extended version as a follow-on.

  early October.
