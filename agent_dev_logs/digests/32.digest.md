## 🎯 Goal
Standardize every service repo's HTTP API to the OpenAPI-followed **meta-contract** of
[`docs/agent_skill_strategy.md`](../../docs/agent_skill_strategy.md) §4 — uniform `GET /info`
(`service`/`version`/`endpoints`/`limits`) + `GET /health` (shallow + `?deep=true`), the §4.4
error-code table, canonical `MAX_UPLOAD_MB`/`ALLOWED_ORIGINS`, version from `para_config.txt` — with
domain endpoints untouched and no committed OpenAPI snapshots (validate the live `/openapi.json`
instead). Enforce it in CI so it can't drift, the same way `para-drift` guards the paradata trio.
